# Excel-Analytics-Platform

A simple Excel Analytics Web App built using the MERN stack.
Users can upload an Excel file, and the application automatically reads the data and converts it into interactive charts and graphs.
# Features
Upload Excel (.xlsx, .xls) files
Automatic conversion of Excel data to JSON
Generate visual charts like:
Bar Chart
Line Chart
Pie Chart
Area Chart
Dynamic graph rendering based on sheet data
Error handling for invalid Excel formats
Responsive & clean UI
Simple backend API for file processing
# Tech Stack (MERN)
Frontend
1. React.js
2. Axios for API requests
3. Recharts / Chart.js for graphs
4. Tailwind CSS / CSS for styling
Backend
1. Node.js
2. Express.js
3. Multer for file upload
4. xlsx for reading Excel files
Database
1. MongoDB
